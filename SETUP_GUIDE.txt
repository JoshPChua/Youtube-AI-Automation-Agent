STEP 1 — CREATE YOUR PROJECT FOLDER

Check if Python is already installed:

  2. Type:  python --version

     Download Python 3.12 → Install it

STEP 6 — GOOGLE SHEETS SERVICE ACCOUNT
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

This lets the agent read/write your Google Sheet without a browser.

  1. Go to: https://console.cloud.google.com
  2. Sign in with your Google account
  3. Click "Select a project" at the top → "New Project"
     Name it: jpeeezy-agent → Create

  4. In the search bar at the top, search: "Google Sheets API"
     Click it → click "Enable"

  5. Search again: "Google Drive API" → Enable that too

  6. Left sidebar → "IAM & Admin" → "Service Accounts"
     → "+ Create Service Account"
     Name: jpeeezy-sheets → Create and Continue → Done

  7. Click on the service account you just created
     → "Keys" tab → "Add Key" → "Create New Key"
     → Choose JSON → Create
     → A file downloads automatically

  8. Rename that downloaded file to:
        gsheets_service_account.json
     Move it into your jpeeezy folder (same folder as the .py file)

  9. Open the JSON file in Notepad — find "client_email"
     Copy that email address (looks like: jpeeezy-sheets@xxx.iam.gserviceaccount.com)

  10. Open your Google Sheet:
      https://docs.google.com/spreadsheets/d/12LWKmgnol-IeHe8Xfm6sVtYMIlvVC6L_qH4hxMwJU_g
      → Click Share (top right)
      → Paste the service account email
      → Set to "Editor"
      → Send

  ✅ Done! The agent can now read and update your sheet.

STEP 7 — YOUTUBE OAUTH (2 CHANNELS)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

You need to do this ONCE for each YouTube channel.

  1. Go to: https://console.cloud.google.com
     (same project you created in Step 6)

  2. Search "YouTube Data API v3" → Enable it

  3. Left sidebar → "APIs & Services" → "Credentials"
     → "+ Create Credentials" → "OAuth 2.0 Client IDs"

  4. If asked to configure consent screen:
     → External → Create
     → App name: Jpeeezy Agent
     → Your email for support
     → Save and Continue through all steps
     → Back to Credentials

  5. "+ Create Credentials" → "OAuth 2.0 Client ID"
     → Application type: Desktop App
     → Name: Jpeeezy Channel
     → Create

  6. Click Download (arrow icon) on the credential you just made
     → Rename the file to: youtube_credentials_jpeezy.json
     → Move to your jpeeezy folder

  7. Repeat steps 5-6 for the second channel:
     → Name: AI Money Channel
     → Download → rename to: youtube_credentials_ai_money.json
     → Move to your jpeeezy folder

  ⚠️ IMPORTANT — Add Test Users:
     Left sidebar → "OAuth consent screen" → "Test users"
     → Add Both your Google account emails
     (the ones connected to each YouTube channel)

  ✅ First time you run the agent, a browser window will open
     asking you to log in. Just click Allow.
     After that it saves the token — no browser needed again.

COMMON ERRORS & FIXES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ❌ "ModuleNotFoundError"
     → Run: pip install -r requirements.txt again

  ❌ "No pending topics found"
     → Open your Google Sheet → make sure some rows have status = "pending"

  ❌ "FileNotFoundError: gsheets_service_account.json"
     → Make sure that file is in the SAME folder as jpeeezy_agent.py

  ❌ "Access denied" on Google Sheets
     → Make sure you shared the sheet with the service account email (Step 6 #10)

  ❌ YouTube auth browser doesn't open
     → Make sure you added your Google account as a Test User (Step 7 ⚠️)

  ❌ "invalid_grant" on YouTube
     → Delete the .pickle token files and run again — it will re-auth
